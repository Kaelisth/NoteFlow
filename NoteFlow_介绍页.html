核心功能

一切你需要的，
没有你不需要的。

精心设计的功能集合，专为日常思考、记录与整理而生。

✒️

富文本编辑器

支持加粗、斜体、下划线、删除线、多级标题、有序与无序列表、引用块、代码块、超链接、表格，以及文字颜色与高亮标注。所见即所得，撤销与重做随时可用。

📊

实时写作统计

底部状态栏实时显示字符数、词数、段落数、行数与预计阅读时长，媲美 Word 的数据指数。

📎

多类型文件嵌入

图片直接嵌入正文预览，PDF、Word、Excel、音视频等文件以卡片形式内联，点击即可下载。

☑️

待办清单

一键插入可勾选的 checkbox 列表，将待办事项直接融入备忘内容，随时标记完成状态。

💾

自动保存

输入后 0.8 秒自动将内容存入 localStorage，关闭页面刷新均不丢失，无需手动保存。

🔍

全文实时搜索

侧边栏搜索框即时过滤，同时检索标题与正文内容，海量笔记中秒速定位目标备忘。

↓

一键导出

将当前备忘导出为独立 HTML 文件，保留所有格式与内嵌图片，可直接用浏览器打开分享。

⇔

可调节侧边栏

拖拽边界自由调整侧边栏宽度，适配不同屏幕与工作流偏好，布局完全由你掌控。

使用方式

三种方式，随你选择。

1
🌐

浏览器直接使用

双击 NoteFlow.html，用 Chrome / Edge / Firefox 打开即可。无需安装任何软件，数据保存在浏览器本地。

2
⚡

Electron 打包 exe

安装 Node.js 后执行 npm install && npm run build:win，生成专业桌面安装包。

3
🐍

Python 单文件 exe

安装 pywebview 和 PyInstaller 后执行打包命令，生成无需依赖的单文件可执行程序。

∞
🔓

完全开放源码

所有源码随 zip 包提供，HTML、main.js、launcher.py 一览无余，随意修改与二次开发。
